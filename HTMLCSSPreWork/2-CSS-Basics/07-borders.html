Borders

The border property allows you to specify the style, width, and color of an elemtn's border

Border Style

The Border-style property can take up to four values (top border, right border, bottom border, and left border). The following border-style value are allowed:

  • dotted: dotted border
  • dashed: dashed border
  • solid: solid border
  • double: double border
  • groove: 3d grooved border, depends on border-color value
  • ridge: 3d ridge border, depends on the border-color value
  • inset: 3d inset border, depends on the borde-color value
  • none: no border
  • hidden: hidden border
  • mixed: you can set a different border sytle for the top, bottom, right, and left

Border Width

VERY IMPORTANT: border-width will not work unless you have border style selected!!

border-width can be set to a specific size (px, em, etc) or you can use one of the following pre-set settings: thin, medium, thick. The border-width property can also take up to four values (top, right, bottom, left

Border Color

VERY IMPORTANT: border-color will not work unless you have border-style set!!

Border-color can be set by the following ways:

  • name: specify a color such as "blue"
  • Hex specify a hex value, like "#eee"
  • RGB specify a RGB value, like "rbb(100,100,100)"
